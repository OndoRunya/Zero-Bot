# Zero-Bot

## How to install ?

### Linux
```
npm install discord.js
npm install canvas
npm install nodemon --save-dev
```
**OR**
```
chmod u+x install.sh
./install.sh
```

### Windows
```
download nodejs https://nodejs.org/en/download/
open windows terminal(cmd)
npm install discord.js
npm install canvas
npm install nodemon --save-dev
```
### MacOS
```
download nodejs https://nodejs.org/en/download/
```

## Run (Available for Windows and MacoOS and Linux)

```
npm run dev
```
